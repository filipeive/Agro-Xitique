O score de crédito é calculado com Programação Funcional — funções puras garantem resultados reproduzíveis e auditáveis.

-- Tipos algébricos do domínio data Contribuicao = Contribuicao { valor :: Double , emDia :: Bool } deriving (Show)
-- Função pura de scoring calcularScore :: [Contribuicao] -> Int -> Int calcularScore cs tam = let pont = length (filter emDia cs) * 10 bonus = if tam >= 10 then 20 else 0 in min 100 (pont + bonus)
-- Teste no GHCi: > calcularScore [Contribuicao 200 True, Contribuicao 200 True, Contribuicao 200 False] 8 20 -- 2 pontuais × 10 = 20

💡 O código Haskell completo está na pasta haskell/ e pode ser executado no GHCi.
